Text-Adventure Game

This unnamed text-adventure game is a project I worked on in late 2021. The goal of the player would be to gather 3 specific objects tagged with 'victory'. The game is played in the console, and different maps could be chosen by loading in different JSON files. A simple combat system exists, along with the capability to pick up and use items like keys or equipment. Here is a link to the GitHub repository.

Racetrack AI

The goal of this project was to set up a simple machine learning program with JavaScript that could drive around the racetrack. I used the Toy Neural Network libarary to create this program. The idea is that a set number of cars is automatically generated, all controlled by the neural network. They are able to detect the walls of the hardcoded racetrack through raytracing, which they can use to get better every round. When all cars died, the new ones are generated based on the best of the last round. Here is a link to the GitHub repository.

Bomb Dodger

This is my latest project, which I've done together with Karsten Veltman. We made a game where the objective is to dodge the explosions of the various bombs spawning in the field. The player character is controlled by tilting the phone, which is done through the accelerometer. The game is also playable against each other in multiplayer, which is done through a combination of the Netcode for GameObjects and Unity Relay libraries. Here is a link to download the game.
